## 🏗️ 技术架构

### 前端技术栈
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS + Headless UI
- **状态管理**: React Hooks + Context API
- **图标**: Heroicons
- **图表**: 自定义图表组件
- **编辑器**: 增强版Markdown编辑器

### 后端技术栈
- **框架**: NestJS (TypeScript)
- **数据库**: PostgreSQL + Prisma ORM
- **认证**: JWT + Passport
- **缓存**: Redis
- **AI集成**: 多提供商API支持
- **文件处理**: 多格式报告生成

### 基础设施
- **容器化**: Docker + Docker Compose
- **负载均衡**: Nginx
- **监控**: Prometheus + Grafana
- **日志**: ELK Stack
- **部署**: 自动化CI/CD
